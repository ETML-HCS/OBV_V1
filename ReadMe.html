🌟 Organisateur de Bureau Virtuel (OBV) 🌟

📋 Description

L'Organisateur de Bureau Virtuel (🖥️ OBV) est une application multiplateforme conçue pour améliorer la gestion et l'organisation des fichiers sur les bureaux virtuels des utilisateurs sous Windows et Linux. Elle automatise la classification des fichiers et crée des espaces de travail personnalisés pour une meilleure efficacité et une expérience utilisateur optimisée.

🚀 Fonctionnalités Clés

1. Classification Automatique des Fichiers 🗂️

  • Identifie les types de fichiers sur le bureau (documents, images, PDF, etc.).
  • Classe les fichiers dans des dossiers ou espaces dédiés basés sur leur type.

2. Création d'Espaces de Travail Dynamiques 🌈

  • Permet la création d'espaces de travail personnalisés.
  • Personnalisation de l'apparence et de la disposition des espaces.

3. Intégration avec l'Explorateur de Fichiers 📁

  • S'intègre avec l'explorateur de fichiers de Windows et le gestionnaire de fichiers de Linux.
  • Gestion facile et intuitive des fichiers.

4. Interface Utilisateur Intuitive 👁️

  • Interface utilisateur simple et conviviale.
  • Options de personnalisation pour les préférences des utilisateurs.

5. Raccourcis Clavier et Commandes Rapides ⌨️

  • Raccourcis clavier pour une gestion efficace des espaces de travail.

6. Sauvegarde et Restauration des Configurations 💾

  • Fonctionnalité de sauvegarde des configurations.
  • Restauration facile sur différents appareils.

7. Compatibilité Multiplateforme 🖥️🐧

  • Compatible avec Windows et Linux.
  • Développée avec .NET 6 pour une performance optimale.

💻 Technologies Utilisées

  • Langage de Programmation: C#
  • Framework: .NET 6
  • Interface Utilisateur: Avalonia
  • Gestion de Version: Git
